SupermarketSim-Windows.zip
==========================

Windows x64 build of "Supermarket Sim".

How to play:
1. Download SupermarketSim-Windows.zip
2. Extract the archive
3. Run SupermarketSim.exe

Controls:
- WASD — move
- Mouse — look
- E — interact
- Esc — exit mode / close UI

PostgreSQL is optional for gameplay.
