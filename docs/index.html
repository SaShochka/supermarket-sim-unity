CREATE TABLE sales (
    product_code TEXT NOT NULL,
    unit_price   INTEGER NOT NULL DEFAULT 15,
    quantity     INTEGER NOT NULL DEFAULT 1,
    total_price  INTEGER GENERATED ALWAYS AS
                 (unit_price * quantity) STORED
);

Возможности

ТерминалЗаказ продуктов, оборудования и найм NPC-кассира.
ВыкладкаПереноска коробок и размещение товаров на полках.
ПокупателиNPC идут по NavMesh: вход → полка → касса.
ЭкономикаКошелёк игрока, продажи и рост магазина.

Быстрый старт

  1. Клонируйте репозиторий и откройте проект в Unity Hub (версия 6000.4.4f1).
  2. Запустите сцену Assets/Scenes/Menu.unity, затем Game.unity.
  3. Управление: WASD, мышь, E — взаимодействие, Esc — выход из режима.
  4. Опционально: установите PostgreSQL и выполните PowerShell-скрипт из Database/postgres/.
